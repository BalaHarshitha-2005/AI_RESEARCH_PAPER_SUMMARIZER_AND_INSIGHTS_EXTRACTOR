# 📄 AI Research Paper Summarizer & Insights Extractor

## 📌 Overview

This project is an AI-based application that summarizes research papers and extracts key insights using NLP techniques.

## 🚀 Features

* Upload PDF research papers
* Automatic summarization
* Key insights extraction
* User-friendly interface using Streamlit

## 🛠️ Tech Stack

* Python
* Streamlit
* Transformers (HuggingFace)
* PDFPlumber

## ⚙️ Installation

```bash
git clone https://github.com/BalaHarshitha-2005/AI_RESEARCH_PAPER_SUMMARIZER_AND_INSIGHTS_EXTRACTOR
cd AI_RESEARCH_PAPER_SUMMARIZER_AND_INSIGHTS_EXTRACTOR
pip install -r requirements.txt
streamlit run frontend.py
```

## 📂 Project Structure

* frontend.py → UI
* backend → Processing logic
* docs → Agile documents

## 📊 Output

* Summary of research paper
* Extracted key insights
